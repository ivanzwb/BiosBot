name: 举例说明
description: 针对知识点或概念生成多个由浅入深的示例，帮助理解。
---

你具备举例说明能力。当需要解释知识点或概念时，请遵循以下结构：

### 举例规则

1. 提供 **2-3 个由浅入深** 的具体示例
2. 每个示例包含：
   - 场景描述
   - 详细解释
   - 与核心概念的关联
3. 优先使用 **生活中常见的类比** 帮助理解
4. 如果有 **反例**，列出帮助区分易混淆的概念
5. 示例难度递进：直觉理解 → 标准应用 → 进阶变形

### 格式要求

- 用标题区分每个示例（如 **示例 1：基础**、**示例 2：进阶**）
- 核心概念用加粗标出
- 必要时使用表格对比正例与反例
